Experiment 1

CSS - Image hover Experiment













About the Experiment :

In this experiment on mouse over on the image , image will zoom out to create an animation. In this experiment I have used images that are available on homepage of my site. I will be applying same feature to my home page . I have used image hover and also aligned position . Designing your web page is very important to make it pretty and viewable.

Body code snippet :


        <div class="thumbnail1"> 

                <div class="image"> 

               <img src="../../images/webImages/experiments.png" id="btnTest" alt="Experiments"> 

               </div> 

            </div >

Description :

In the above code snippet ,I have used two style class to make image stay in the particular position and to make image animation work accordingly . Thumbnail1 class will set the image position to particular axis on the screen. Image class will create animation on mouse hover on the image.

Style code snippet

.image {

            width: 100%;

            height: 100%;

        }

            .image img {

                -webkit-transition: all 1s ease; /* Safari and Chrome */

                -moz-transition: all 1s ease; /* Firefox */

                -o-transition: all 1s ease; /* IE 9 */

                -ms-transition: all 1s ease; /* Opera */

                transition: all 1s ease;

            }

            .image:hover img {

                -webkit-transform: scale(1.25); /* Safari and Chrome */

                -moz-transform: scale(1.25); /* Firefox */

                -ms-transform: scale(1.25); /* IE 9 */

                -o-transform: scale(1.25); /* Opera */

                transform: scale(1.25);

            }
