About the Experiment

Date and time are very important when we need to receive data on stipulated time also when we are using and API which sends responce for selected time date and time fields play a major role.

In this experiment I have used HTML5 date and time input type and did following validations :
1) Start Date should be less than end date :
Alert Message is shown to select the end date greater than start date

2) When Same date is selected for both start date and end date and also start time is greater than end time.
Alert Message is shown to select the end time greater than start time

3) All fields in the experiment are mandatory , Incase of fail
Alert message is shown to select all the fields.
