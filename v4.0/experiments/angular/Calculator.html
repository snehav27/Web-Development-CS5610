About the Experiment

After completing my first experiment , I have done another experiment on AngularJS without any reference . I have designed simple calculator using AngularJS to calculate simple operation like Addition,Substraction,Multiplication and Division.

Library for Angular JS

      < script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.2.15/angular.min.js"/ >

Code Snippet View:

                <div ng-controller="CalculatorController"> 

                <input type="number" name="number1" ng-model="number1" placeholder="Enter Number 1" /> 

                <p ng-bind="output"></p>
Description :

Above code snipper is html code :
1) ng-controller : Directive defines the application controller and it is the controller object.
2) ng-model : Directive binds the value of HTML controls (input, select, textarea) to application data. To identify the object.
3) ng-bind : directive binds application data to the HTML view.

Code Snippet Controller :

                var num1 = $scope.number1; 

                var num2 = $scope.number2; 

                var selectedOption = $scope.selectBox;
Description :

The $scope of the controller is the application (the HTML element) it is referred from. Here we are fetching customer input
