About the Experiment

AngularJS is developed by Google and this framwork so easy to use and developer friendly.Lot of developers started using it now. In this experiment AngularJS is used to show table data and add data into table when enetered by user. ContactController is the controller for this add operation. Onclick of add button , details of contact (name, email and number ) are sent to controller and automatically added into table without loading page. With colabration of ajax and angular js we will be able to DB operation without loading the page . Angular JS can also be used when we fetch data from other API as an object.

Code Snippet View:

Note: Here I have given only name example

          < div ng-controller="ContactController" > 

          < input type="text" name="name" ng-model="newcontact.name" placeholder="Name of the Contact" /> 

                < div  ng-repeat="contact in contacts" > 
                
                           contact.name 

                       < /div >
Description :

Above code snipper is html code :
1) ng-controller : Directive defines the application controller and it is the controller object.
2) ng-model : Directive binds the value of HTML controls (input, select, textarea) to application data. To identify the object.
3) ng-repeat : Directive repeats an HTML element . When lot of data needs to be shown we can use ng-repeat to iterate the data and display in the screen.
4) contact.name is ouput pasted on the screen when user enters name and add button is clicked.
