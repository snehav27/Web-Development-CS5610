Using Marquee in HTML

Hover To Stop This Text....

About the Experiment

The HTML < marquee> element is used to insert a scrolling area of text. Marque is an intresting element in html. Marque can be used for scrolling news, displaying any content that can be scrolled. We can add attributes to control the scrolling effect. I have used scrolling effect to show todays fuel price in my final project.

Code

   
            
         <marquee behavior="scroll" direction="left" onmouseover="this.stop();"  

onmouseout="this.start();">Hover To Stop This Text....</marquee>
Description :

behavior Sets how the text is scrolled within the marquee. Possible values are scroll, slide and alternate. If no value is specified, the default value is scroll.
Direction : will indicate the direction in which the text is scrolling .

start : Starts scrolling of the marquee.
stop : Stops scrolling of the marquee.

On mouse out is when user doesnt point with mouse on the scrolling text.
On mouse over is when user point with mouse on the scrolling text.
