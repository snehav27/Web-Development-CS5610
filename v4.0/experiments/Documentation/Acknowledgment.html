Goal: The objective of this web development project is to design a website, and to allow user to fetch information of cars, get the information about the environment friendly car. Login and user will be able to navigate to others profile and discuss about cars in the forum.

System Features:

1) CarInfo:
Car info page will provide details of the car series information, car pictures,dealers info near by,car price,equipment and car review information.
2) Go Green:
Go Green tab will allow user to select the appropriate car based on budget , mileage and cars which have more green house score .
This tab also gives the information current days fuel price.
3) Map Info
This page will allow users to fetch near by car parking and car service centers information.
4)News Info
T This page allows user to see recent news created by admin.
5) Login
Login page allows user to login and user will be able to see his own dashboard.
6) Forum
Forum Will allow user to sell their cars and discuss on car topics .
