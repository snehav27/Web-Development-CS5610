Car Info Exaplanation :

This page allows user to fetch information about the cars from edmunds API .


Base Technologies used:
1) Angular JS
2) ASPX (to fetch data from the DB for dropdowns)
3) JSON format

Responsive:
Fully Responsive , no Bootstrap Used.
