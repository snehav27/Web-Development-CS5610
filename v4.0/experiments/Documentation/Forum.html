Forum Exaplanation :

Every website has forum, where people come and discuss about website related topics. In my website, I have created forum so that users will be able to discuss about the cars, sell their cars, off topics.
This Module consists of three sections 1)Forum Page consists of Topics
2)Posts Page associated to topics.
3)Post page containing comments posted by other users.
