Go Green Exaplanation :

This page is for user to fetch Green Cars . Users will be able to fetch green cars . User can select a car and get details of carbon dioxide and green scores. They can also fetch the data of how much mileage each car provides in highway, city and both combined. user has a flexibilty to fetch these data in 3 ways .
1)By Car model name.
2)By Car Mileage input.
3)By Car Green score.

Todays Fuel Price :

This is the information fetched from a web service to provide current fuel price .


Base Technologies used:
1) ASPX (to fetch data from the DB for dropdowns And table output)
2) XML (webservice)

Responsive:
Fully Responsive , no Bootstrap Used.
