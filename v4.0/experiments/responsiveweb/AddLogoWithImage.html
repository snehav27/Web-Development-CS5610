About the Experiment :

This experiment is continuation of Experiment2 of Responsive Image.

In this experiment I have tried following responsive : changing Image to text according to screen size.

CSS code snippet : Media Query :

       .logo { 

                background-image: url('../../images/webImages/header.jpg');

                background-size: 100%;

                width: 100%;

                padding-top: 20.8%;

                height: 0;

                text-indent: -9999px;

            }

HTML View Code Snippet:

        <h1 class="logo">Car Point </h1>
Description :

In this scenerio Image changes and converts to Text inside H1 after particular screen size. Class logo will get the properties in css .logo and apply properties in that H1 where image is overlays on text, once the screen width reduces it will make the Image disappear and only Text inside H1 will be viewed.

1) background-image : gets the image which will be overlayed on the text.
2) text-indent: -9999px; : will make text to hide behind image until the screen size is reduced.
