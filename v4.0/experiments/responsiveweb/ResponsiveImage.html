Responsive Image

This experiment is to know about Resposive image and how @media screen is used

I have took an image of flowers in boston which is clicked by me :) , to demonstrate responsive image.

Code snippet example for Responsive Image

          
        @media screen and (max-width: 920px) {
            #image1 {
                width: 430px;
            }
        }

max-width : (here pixels are increased according to the screen size of the device) width : Pixels of the image to be shown according to the max-width By creating a resposive images we will be able shrink the image according to the screen size . So Resposive image can be viewed on any type of the devices.
