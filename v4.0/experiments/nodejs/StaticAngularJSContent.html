About the Experiment

In this experiment I have hosted static angular JS content in the Open Shift. To get gist on how to host in Open Shift and deploy the content into git . More info on AngularJS : Angular JS Experiments

Angular JS Content : Code Snippet View:


        <tr> <ng-repeat="carCtrl in carDetails"> 

        <td>{{carCtrl. user}}</td> 

        <td>{{carCtrl. carname}}</td> 

        <td>{{carCtrl. review}}</td> 

                        </tr>
Description :

Above code is view page where user , carname , review is fetched from the carDetails object which is set by controller in the javascript page.

Angular JS Content : Code Snippet Controller:

            function CarReviewCtrl($scope,$http) { 

    console.log("angular from services");

    var carReview1 = {

        user: "sneha",

        carname: "bmw",

        review: "this is an awsome car"    

    };

    var carReview2 = {

        user: "swetha",

        carname: "audi",

        review: "this is an cool car to have"

    };

    var carReviews = [carReview1, carReview2];

    $scope.carDetails = carReviews;

}
Description :

Above code snippet will send a static page responce to the client who have accessed the link. In the above code controller will set static content declared in carReview1 and carReview2 and assign it to the Cardetails which is refered in the html page . $scope.carDetails will set the scope variable which will reflect in the html page.
