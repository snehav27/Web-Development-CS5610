Code Snippet

       
            app.get('/getallcarreview', function (req, res) { 

                    db.carreview.find(function (err, data) {

                    res.json(data);

                  });

             });
Description :

Above code snippet will send a json responce from MongoDB to the client who have accessed the link .4 Table Contains Details of car Review , when the /getallcarreview link is cliked it will fetch all of the car review from carreview table and sends responce in JSON.
1) find is the keyword used in the mongo db to fetch all the data
2) findOne is the keyword used to fetch data for one parameter.
